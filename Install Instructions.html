Tile Swap for Connections

Install instructions

  1. Move this folder somewhere safe Chrome needs the folder to stay in place after installing. Put it in your Documents folder (or anywhere you won't accidentally delete it).
  2. Open Chrome's Extensions page Click the puzzle-piece icon in the top-right corner of Chrome, then click Manage Extensions at the bottom of the menu.
  3. Turn on Developer Mode In the top-right corner of the Extensions page, switch the Developer mode toggle ON.
  4. Click "Load unpacked" A button will appear in the top-left. Click it, then select the folder that contains this file.
  5. You're done! Open (or refresh) NYT Connections and drag tiles to rearrange them before guessing.

Notes

The extension only runs on the NYT Connections page — no other sites are affected.

To uninstall, go back to Extensions (puzzle-piece icon → Manage Extensions), find Tile Swap for Connections, and click Remove.

You can also reach the Extensions page by typing chrome://extensions in the address bar.
